A small index repository that links the two practical ways to use **ClickUp with OpenClaw**:

1. **Native OpenClaw ClickUp tools**
2. **Manual MCP integration for ClickUp**

This repo is the front door. If you forget which route to use later, start here.

## Which route should I use?

- Use the **native route** if your OpenClaw environment already exposes `clickup__*` tools.
- Use the **MCP route** if you want a reusable manual MCP setup saved in OpenClaw config.
